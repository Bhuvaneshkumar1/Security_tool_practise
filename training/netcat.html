ENUMERATION & MAPPING

Extracting meaning from discovered services

1. What Is Enumeration?

Enumeration is the phase where an attacker actively interacts with services to understand how they behave, respond, and fail.

  • What inputs are accepted?
  • What responses are returned?
  • What information is leaked?

2. Why Netcat Matters in Enumeration

Netcat (nc) is a low-level networking utility that moves raw bytes between STDIN and network sockets. 

STDIN → Network Socket → STDOUT

It does not understand protocols — which makes it ideal for observing raw behavior.

3. Client vs Listener (Concept)

Listener Mode

nc -lvp 4444

Client Mode

nc 127.0.0.1 4444
In this platform, listener execution is intentionally disabled for safety.

4. Banner Grabbing & Protocol Interaction

nc target.com 80
GET / HTTP/1.1
Host: target.com

This reveals server software, versions, and input handling.
